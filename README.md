## How to install:
```cd /etc/conky/```
and replece conky.conf

## How to customize:
#### change font: 
  - change all "FreeMono", to your own font name

#### change alignment: 
  - change "alignment" variable to desired one, check http://conky.sourceforge.net/config_settings.html for reference
